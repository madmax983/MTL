# seal_digit_sum_base

Tier: tier3  |  Category: arithmetic

Given two integers n >= 0 and b >= 2, write n in base b (the standard positional representation with digits 0..b-1, no leading zeros except for n=0 which is the single digit 0) and return the sum of those base-b digits.

Signature: `f(n, b) -> int`. Stack input (mtlrun prefix): `0 2 ...` — arguments are pushed left to right, so the last argument is on top of the stack when the program starts.
