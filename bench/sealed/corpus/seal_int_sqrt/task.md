# seal_int_sqrt

Tier: micro  |  Category: arithmetic

Given an integer n >= 0, return the integer square root of n: the largest integer r such that r*r <= n. This is the floor of the real square root of n.

Signature: `f(n) -> int`. Stack input (mtlrun prefix): `0 ...` — arguments are pushed left to right, so the last argument is on top of the stack when the program starts.
