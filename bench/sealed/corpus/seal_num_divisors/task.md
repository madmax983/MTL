# seal_num_divisors

Tier: micro  |  Category: arithmetic

Given an integer n >= 1, return the number of positive integers that divide n exactly (with no remainder), including 1 and n itself.

Signature: `f(n) -> int`. Stack input (mtlrun prefix): `1 ...` — arguments are pushed left to right, so the last argument is on top of the stack when the program starts.
