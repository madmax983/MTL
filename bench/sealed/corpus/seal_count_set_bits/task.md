# seal_count_set_bits

Tier: micro  |  Category: arithmetic

Given an integer n, take its absolute value and return the number of 1-bits in the binary representation of that absolute value (its population count). The absolute value of 0 has zero 1-bits.

Signature: `f(n) -> int`. Stack input (mtlrun prefix): `0 ...` — arguments are pushed left to right, so the last argument is on top of the stack when the program starts.
